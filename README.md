# CGPA Calculator

A C++ program that calculates the CGPA of a student based on course grades and credit hours.

## Features

* Input course names
* Input grades and credit hours
* Calculate total grade points
* Calculate final CGPA
* Display detailed course information

## Technologies Used

* C++
* Object-Oriented Programming
* Standard Template Library (STL)

## Internship

This project was developed as part of the CodeAlpha C++ Programming Internship.
